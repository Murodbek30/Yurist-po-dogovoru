# Дизайн Система (Design System)

## Общая концепция
Проект будет использовать премиальный, строгий, но современный 'Dark Mode' стиль. Использование эффекта "Glassmorphism" (эффект матового стекла) для карточек и форм придаст интерфейсу глубину и высокотехнологичный вид (что особенно важно для сервиса, использующего искусственный интеллект для анализа).

## Цветовая палитра
- **Background**: `#0F172A` (глубокий темно-синий/сланцевый цвет).
- **Surface/Cards**: Тонкий полупрозрачный белый/синий с блюром (например, `rgba(255, 255, 255, 0.05)`).
- **Primary Accent (Telegram/Trust)**: `#3B82F6` (Electric Blue).
- **Secondary Accent (WhatsApp/Success)**: `#10B981` (Emerald Green).
- **Warning/Alert**: `#EF4444` (Красный для предупреждений о рисках).
- **Text Primary**: `#F8FAFC` (Почти белый).
- **Text Secondary**: `#94A3B8` (Приглушенный сланцевый).

## Типографика
- **Шрифт**: `Inter` или `Outfit` (от Google Fonts).
- **Заголовки**: Смелые, с легким градиентом для акцентирования ключевых слов (например, Анализ, Защита).

## Анимации
- Плавные `hover` эффекты (увеличение масштаба карточек, подсветка рамок).
- Плавное появление (Fade-in) элементов при загрузке.
- Индикатор "печатает..." (typing bubble) для ИИ-ассистента в чате.
- Состояния загрузки (спиннеры или скелетные экраны) при анализе фото (OCR) и генерации документов.

## Компоненты UI

### Кнопки
Кнопка должна быть большой, заметной, с градиентом и мягкой тенью (`box-shadow`), чтобы привлекать клик. Для функций ИИ (например, "Рассчитать срок") должны быть компактные кнопки (chips) в интерфейсе чата.

### Компоненты Чата (Web-интерфейс)
- **Message Bubble (AI)**: Выровнен по левому краю, использует Surface/Cards цвет, текст Text Primary. Включает структурированные блоки (Анализ, План, Инструменты, Совет).
- **Message Bubble (User)**: Выровнен по правому краю, использует Primary Accent (Electric Blue), текст Text Primary.
- **Upload Zone**: Drag-and-drop область для загрузки фото/сканов с иконкой и пунктирной рамкой в стиле Glassmorphism.

### Инфраструктурные панели
- **Левое меню**: История дел/споров.
- **Правое меню**: Панель превью документов и редактирования сгенерированных файлов (`generate_document`).
