# Правовой Щит (Legal Shield)

Юридический ИИ-ассистент для граждан, помогающий в решении споров с государственными органами.

## О проекте
"Правовой Щит" — это веб-приложение, предоставляющее пользователям эмпатичного и грамотного ИИ-ассистента. Ассистент анализирует ситуации, предлагает План "Минимум", генерирует документы и рассчитывает сроки, общаясь в спокойной и поддерживающей манере.

## Ключевые возможности
* **Интеллектуальный диалог:** Использование алгоритма "Даю + Спрашиваю" для точного сбора информации без излишнего стресса для пользователя.
* **Генерация документов (`generate_document`):** Создание точных требований, заявлений и жалоб (с наступательным или нейтральным тоном в зависимости от ситуации).
* **Анализ текста (`analyze_text`):** Проверка писем на ошибки, наличие четких требований и контактов.
* **Распознавание документов (`analyze_image`):** Загрузка фото, OCR-анализ и выгрузка в Word для редактирования.
* **Расчет сроков (`calculate_deadline`):** Определение точных сроков подачи с учетом "бюрократических" задержек.

## Технологии
- HTML5, Vanilla CSS (с использованием CSS Variables, Flexbox, Grid)
- Эффекты: Glassmorphism, Dark Mode

## Дизайн
Приложение выполнено в премиальном строгом Dark Mode стиле с использованием синих и зеленых акцентов, плавных анимаций и матового стекла (glassmorphism), что подчеркивает надежность и высокотехнологичность.
